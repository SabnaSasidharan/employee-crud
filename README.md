# EMPLOYEE DETAILS

## Description

Employee details view , edit, update, delete and insert

## Getting Started

### Dependencies

* make sure there is an autoload file in vender folder

### Installing

* Download program in htdocs
* Created a table in phpmyadmin where database name is laravel, username is root, password is nill.


### Executing program

* In local server like Xampp is used for running the program (ie; http://127.0.0.1:8000/employee) in localhost
* First display the employee details in a paginated view
* From there we can add new employee through a button click
* We can update and delete employee from the same displayed page.
